# Alpine Light Prompt Guidelines

Use this when generating UI copy, component styles, or visual prompts for the Alpine theme in light mode.

## Core Identity

- Mood: pale mountain air, glass panels, black-green text, one vivid orange signal.
- Tone: clear, premium, restrained, and calm.
- Avoid: dense comic framing, beige nostalgia, multicolor decorative accents, long explanatory UI text.

## Locked Color Roles

- Main1 (page background): `#dfe8eb`
- Main2 (panel background): `#f2f6f4`
- Support (elevated surfaces): `#9aa7a4`
- Accent (primary saturated color): `#ff3b12`
- Text: `#101816`

## Typography Direction

- Headings: tall editorial serif using `Newsreader` first.
- Body/UI: `Manrope`, `Segoe UI`, sans-serif for Vietnamese-safe interface text.
- Labels: compact uppercase is allowed for metadata only.

## Component Direction

- Panels: soft glass surfaces with thin borders and restrained shadows.
- Active navigation: one subtle outline plus orange accent wash.
- Inactive navigation: no visible outline unless hovered or focused.
- Actions: orange pill or compact glass button; keep wording short.

## Motion Direction

- Use restrained 140-220ms transitions.
- Prefer opacity, border-color, and small translate changes.
- Avoid looping motion and bouncing effects.
