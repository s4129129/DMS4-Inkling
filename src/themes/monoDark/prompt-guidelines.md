# Monochrome Dark Prompt Guidelines

Use this when generating UI copy, component styles, or visual prompts for the Monochrome theme in dark mode.

## Core Identity

- Mood: cinematic control-room, focused and minimal.
- Tone: concise, technical, and calm.
- Avoid: colorful accents, noisy texture, or excessive glow.

## Locked Color Roles

- Main1 (page background): `#16181a`
- Main2 (panel background): `#22262a`
- Support (elevated surfaces): `#4f565d`
- Accent (actions/highlights): `#d9e0e6`
- Text: `#f4f6f8`

## Typography Direction

- Headings: compact modern sans.
- Body/UI: neutral sans optimized for dense information.
- Maintain strict contrast and consistent spacing grid.

## Component Direction

- Panels: layered dark neutrals with clear edges.
- Actions: cool-light accent reserved for key intent only.
- Popups/tooltips: sharp, readable, no decorative copy.

## Motion Direction

- Subtle transitions only (130-200ms).
- Use fade + slight translate; avoid playful movement.
- Preserve stable focus during interactions.

## Prompt Scaffold

"Design a [component/screen] in the Inkling Monochrome Dark theme. Use Main1 `#16181a`, Main2 `#22262a`, Support `#4f565d`, Accent `#d9e0e6`, and Text `#f4f6f8`. Keep the composition minimal and high-contrast, with neutral sans typography, crisp structural spacing, and restrained 130-200ms transitions."
