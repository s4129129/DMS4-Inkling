# Alpine Dark Prompt Guidelines

Use this when generating UI copy, component styles, or visual prompts for the Alpine theme in dark mode.

## Core Identity

- Mood: black glass interface over cold mountain stone and mist.
- Tone: quiet, premium, sparse, and data-focused.
- Avoid: comic outlines, parchment warmth, many accent colors, heavy explanatory UI text.

## Locked Color Roles

- Main1 (page background): `#050706`
- Main2 (panel background): `#0b0f0d`
- Support (elevated surfaces): `#1b211f`
- Accent (primary saturated color): `#ff3b12`
- Text: `#f4f2ec`

## Typography Direction

- Headings: tall editorial serif using `Newsreader` first.
- Body/UI: `Manrope`, `Segoe UI`, sans-serif for Vietnamese-safe interface text.
- Labels: compact uppercase is allowed for metadata only.

## Component Direction

- Panels: translucent dark glass with soft borders, not thick outlines.
- Active navigation: one subtle outline plus orange accent wash.
- Inactive navigation: no visible outline unless hovered or focused.
- Actions: orange pill or compact glass button; keep wording short.

## Motion Direction

- Use restrained 140-220ms transitions.
- Prefer opacity, border-color, and small translate changes.
- Avoid looping motion and bouncing effects.
