# Default Light Prompt Guidelines

Use this when generating UI copy, component styles, or visual prompts for the Default theme in light mode.

## Core Identity

- Mood: bold comic shop counter, neutral page base, energetic but readable.
- Tone: punchy, direct, and dashboard-focused.
- Avoid: soft parchment styling, washed-out color blends, or decorative blur.

## Locked Color Roles

- Main1 (page background): `#E5E5E5`
- Main2 (panel background): `#F7F7F7`
- Support (elevated surfaces): `#FFCF01`
- Accent (actions/highlights): `#DE2915`
- Text: `#000000`

## Typography Direction

- Headings: condensed comic/display energy using Segoe UI Black/Arial Black with full Vietnamese glyph support.
- Body/UI: sturdy sans with strong weight and crisp contrast.
- Keep letter spacing at `0`; use uppercase through text transform only.

## Component Direction

- Panels: square corners, black borders, hard-offset shadows.
- Actions: yellow blocks with black text when readability matters; use red only for small accents.
- Popups/tooltips: compact, high contrast, no ornamental copy.

## Motion Direction

- Use quick 120-180ms transitions.
- Prefer border, outline, and tiny translate changes.
- Avoid looping or bouncy animation.
